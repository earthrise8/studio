# **App Name**: Fitropolis

## Core Features:

- User Authentication: Securely authenticate users with email/password to protect personal health data.
- Personalized Dashboard: Display a summary of daily calorie intake vs. goal, 'Calories In' and 'Calories Out', quick actions, and expiring pantry items.
- Virtual Pantry Management: Allows users to track food items with name, quantity, expiration date, and category.
- Daily Logging: Log meals with name, calories, protein, carbs, and fat; log activities with name, duration, and calories burned.
- AI Recipe Tool: Suggest recipes based on pantry contents and allow importing from URLs, using AI as a tool to extract recipe details.
- Recipe Collection: Display saved recipes with name, description, ingredients, and instructions.
- AI Health Advisor: Provide personalized health advice based on user logs and stated goals, generating actionable goals and awarding achievements for completed goals.

## Style Guidelines:

- Primary color: Seafoam Green (#7CFFB2) for primary buttons and accents, conveying a fresh and healthy vibe.
- Background color: Soft Mint (#E2F9F0) for the main app background, creating a calming and encouraging environment.
- Accent color: Coral (#FF7F50) for important call-to-action buttons, drawing user attention for key actions.
- Headlines: 'Poppins' (sans-serif) for headlines providing a modern and approachable feel.
- Body: 'PT Sans' (sans-serif) for body text ensuring readability and a clean design.
- Use LayoutDashboard for Dashboard, Warehouse for Pantry, and ChefHat for Recipes, creating visual associations.
- Implement a clean, card-based layout with a fixed sidebar on desktop that collapses into a hamburger menu on mobile, for a responsive experience.
